This is the coursework of Joe Derrett for CSC8503.
Controls:
W: Forward
A: Left
S: Back
D: Right
E: Drop held glass
To pick up the glass, walk into it.
To win, drop the glass on the blue win box at the end of the course
Rotate mouse to move the camera.
This project aims to implement a physics based game into a supplied codebase.
That initial supplied codebase was created by Newcastle University (Made by Rich Davison)
Some AI was used to fix a few issues (mainly for networking)
Otherwise the game is my code unless otherwise stated. 
The map design takes some design ideas from the TV series Total Wipeout (BBC, Original run 2009-2012)
